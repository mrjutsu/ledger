## Contributing

If you wish to contribute to the project, follow these instructions:

- Fork the project.
- Give your branch a descriptive name, e.g.:
-   `feature/my-awesome-feature`
-   `bugfix/issue-fix`
-   `improvement/some-optimization`
- Pull `main` into your branch.
- Make a Pull Request.
